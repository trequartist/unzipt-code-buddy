# ContentQ Keyboard Shortcuts

## Quick Navigation

| Shortcut | Action |
|----------|--------|
| `Ctrl + 1` (or `⌘ + 1` on Mac) | Go to Create workspace |
| `Ctrl + 2` (or `⌘ + 2` on Mac) | Go to Strategy workspace |
| `Ctrl + 3` (or `⌘ + 3` on Mac) | Go to Intelligence workspace |
| `Ctrl + 4` (or `⌘ + 4` on Mac) | Go to Hub workspace |

## Command Palette

| Shortcut | Action |
|----------|--------|
| `Ctrl + K` (or `⌘ + K` on Mac) | Open command palette |
| `Escape` | Close command palette / dialogs |

## Editing & Actions

| Shortcut | Action |
|----------|--------|
| `Ctrl + S` (or `⌘ + S` on Mac) | Save current content |
| `Ctrl + /` (or `⌘ + /` on Mac) | Focus assistant panel |

## Editor Shortcuts

When in the content editor:

| Shortcut | Action |
|----------|--------|
| `Ctrl + B` (or `⌘ + B` on Mac) | Bold text |
| `Ctrl + I` (or `⌘ + I` on Mac) | Italic text |
| `Ctrl + U` (or `⌘ + U` on Mac) | Underline text |

## Accessibility Features

### Screen Reader Support
- All interactive elements have proper ARIA labels
- Skip-to-content links available
- Semantic HTML structure throughout

### Keyboard Navigation
- `Tab` - Navigate forward through interactive elements
- `Shift + Tab` - Navigate backward
- `Enter` - Activate buttons and links
- `Space` - Toggle checkboxes and switches
- Arrow keys work in dropdowns and lists

### Visual Accessibility
- High contrast mode support
- Resizable text (up to 200% zoom)
- Color-blind friendly palette
- Focus indicators on all interactive elements

## Mobile Gestures

- Swipe from right edge - Open assistant panel
- Pull down to refresh - Reload current page
- Long press - Open context menu

## Pro Tips

1. **Command Palette Power**: Use `Ctrl + K` to quickly search and navigate anywhere in the app
2. **Quick Save**: Hit `Ctrl + S` frequently to ensure your work is saved
3. **Assistant Shortcut**: Press `Ctrl + /` to instantly start chatting with the AI assistant
4. **Workspace Switching**: Use `Ctrl + 1-4` to jump between workspaces without using the mouse

## Customization

Keyboard shortcuts respect your system preferences and can work with both Control (Windows/Linux) and Command (Mac) modifiers.
